## 1. アプリの概要

LyricsJP（以下「本アプリ」）は、Android端末で再生中の音楽を検知し、歌詞と日本語訳を表示するアプリです。

## 2. アクセスする情報

本アプリは、利用者がAndroid設定で明示的に許可した「通知へのアクセス」を使用します。この機能から、SpotifyまたはApple Musicが公開する以下の再生情報だけを処理します。

- 曲名
- アーティスト名
- アルバム名
- 曲の長さと再生位置
- 再生／一時停止状態
- 再生元アプリの識別子

本アプリは、対象音楽アプリ以外の通知内容を保存または送信する目的では使用しません。

## 3. 外部サービスへの送信

歌詞検索のため、曲名、アーティスト名、アルバム名、曲の長さをLRCLIBへ送信します。同期歌詞が見つからない場合は、同じ情報を複数の公開歌詞プロバイダーを横断するlrc muxへ送信します。通常歌詞も見つからない場合は、曲名とアーティスト名をLyrics.ovhへ送信します。

利用者が翻訳を有効にした場合、取得した歌詞を、利用者が選択した次の翻訳サービスへ送信します。

- OpenAI API
- Google Gemini API
- DeepL API

送信先サービスによるデータ処理には、それぞれの利用規約およびプライバシーポリシーが適用されます。本アプリの開発者が運営するサーバーへ、これらの情報を送信することはありません。

## 4. APIキーと端末内データ

利用者が入力したAPIキーはAndroid Keystoreを利用して端末内で暗号化します。曲情報、歌詞、翻訳結果および設定は端末内に保存されます。

保存データは、Androidのアプリ情報画面からストレージを消去するか、本アプリをアンインストールすることで削除できます。

## 5. 第三者提供、広告、解析

本アプリは、広告SDK、アクセス解析SDK、開発者独自のサーバーを使用しません。歌詞検索および利用者が選択した翻訳処理に必要な送信を除き、利用者データを販売または第三者へ提供しません。

## 6. セキュリティ

外部サービスとの通信にはHTTPSを使用します。APIキーはAndroid Keystoreを使用して暗号化します。ただし、インターネット通信や端末上の保存について絶対的な安全を保証することはできません。

## 7. 子どもの利用

本アプリは子どもを対象として設計していません。Google Playで設定する対象年齢と本アプリの実際の内容が一致するよう管理します。
